Outils texte Unix pour la bioinformatique : sort, cut, sed, awk

Cette page suit le style demandé et propose un fil conducteur pratique pour manipuler des fichiers tabulaires et formats biologiques (FASTA, GFF/GTF, VCF) avec les outils Unix de base. Les exemples sont minimalistes, reproductibles et prêts à copier-coller.

Objectifs

  • Savoir trier et fusionner des grandes tables (sort, uniq, join).
  • Extraire / réordonner des colonnes (cut, paste, column).
  • Éditer des flux texte (sed) et réaliser des filtrages/calculs (awk).
  • Chaîner ces outils avec les redirections, pipes et métacaractères Bash.
  • Appliquer ces techniques à des cas réels en bioinformatique.

2. sort

Options clés

OptionSignificationExemple
-k i[,j]Clé de tri (colonnes i..j)sort -k1,1
-tSéparateur (par défaut espace)-t $'\t' pour TSV
-nNumériquesort -n -k3,3
-rOrdre décroissantsort -r
-uUnique (garde la 1ère)sort -u
--parallel=64Threadssort --parallel=64

Exemples 

# Tri TSV par gène (col1) puis par score (col3 numérique décroissant)
sort -t $'\t' -k1,1 -k3,3nr genes.tsv > genes.sorted.tsv

# Supprimer les doublons exacts (lignes identiques)
sort file.txt | uniq > file.uniq.txt

# Compter les occurrences (histogramme)
sort samples.txt | uniq -c | sort -nr | head

3. cut, paste, column

CommandeUsageExemples
cut Extraire colonnes/bytes 
# Colonnes 1 et 3 (TSV)
cut -f1,3 table.tsv

# Colonnes non contiguës
cut -f1,5-7 table.tsv
paste Fusionner colonnes (côte à côte) 
# Concaténer deux fichiers ligne à ligne
paste -d $'\t' a.tsv b.tsv > ab.tsv
column Aligner joliment pour lecture 
column -t -s $'\t' table.tsv | less -S

4. sed - Éditeur de flux

Patrons utiles

ActionCommande
Remplacer globalementsed -E 's/ancien/nouveau/g'
Supprimer lignessed '/^#/d'
Conserver lignes (inverser)sed -n '/regex/p'
Modifier séparateursed 's/,/\t/g'
Capturessed -E 's/chr([0-9]+)/\1/'

Exemples

# Nettoyer commentaires et lignes vides
sed -E '/^\s*#/d; /^\s*$/d' data.tsv > clean.tsv

# Normaliser IDs FASTA (garder 1er token)
sed -E 's/^>([^ \t|]+).*$/>\1/' seqs.fasta > seqs.norm.fasta

5. awk - Filtrer, calculer, agréger

ConceptExplicationExemples
Champs $1, $2, … (séparés par -F) 
# TSV : montrer gène et moyenne de colonnes 3..5
awk -F '\t' '{m=($3+$4+$5)/3; print $1, m}' genes.tsv
Filtres Expression vraie → imprime la ligne 
# pval < 1e-5 et logFC >= 1
awk -F '\t' 'NR==1 || ($5<1e-5 && $6>=1)' de.tsv > sig.tsv
Agrégation Associatifs pour regrouper/compter 
# Compter par catégorie (col2)
awk -F '\t' '{c[$2]++} END{for(k in c) print k, c[k]}' table.tsv | sort -k2,2nr
Reformatage Construire des colonnes 
awk -F '\t' 'BEGIN{OFS="\t"} {print $2, $1 ":" $3 "-" $4}' peaks.tsv
Astuce performance : pour de très gros fichiers, combinez LC_ALL=C, sort --parallel, et des pipes pour éviter des écritures intermédiaires.

6. Cas bioinfo guidés

6.1 FASTA : dédoublonner et compter

# Garder 1er token des en-têtes
sed -E 's/^>([^ \t|]+).*$/>\1/' seqs.fasta |
awk 'BEGIN{RS=">"; ORS=""} NR>1{gsub(/\n/,"",$2); print ">"$1"\n"$2"\n"}' |
awk 'BEGIN{RS=">"; ORS=""} NR>1{if(!seen[$1]++){print ">"$1"\n"$2"\n"}}' > seqs.uniq.fasta

# Compter les longueurs
awk 'BEGIN{RS=">", ORS=""} NR>1{len=length($0)-length($1)-1; print $1"\t"len"\n"}' seqs.uniq.fasta |
sort -k2,2nr | column -t

6.2 GFF/GTF : gènes les plus longs

grep -v '^#' annot.gff3 |
awk -F '\t' '$3=="gene"{len=$5-$4+1; print $1,$4,$5,len,$9}' |
sort -k4,4nr | head | column -t

6.3 VCF : variants à haute qualité

grep -v '^##' variants.vcf |
awk 'BEGIN{FS=OFS="\t"} NR==1 || ($6>=200 && $5 ~ /A|T|C|G/)' |
cut -f1-8 | column -t | less -S

7. Exercices
